# 영화 추천 서비스

## How to use

Frontend 
```
# move to forntend folder
npm install 
npm run serve
```

Backend
```
# move to backend folder
python manage.py loaddata {{ data }}
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
